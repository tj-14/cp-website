# Project Requirements and Context

## Goal
- Turn incomplete content in my book draft into an interactive educational website
- Target audience: Thai high school students preparing for Computer Olympiad camps

## Design Inspiration
- Primary: https://cp-algorithms.com/index.html (clean, minimal, content-focused)
- Reference: https://usaco.guide (but keep it simpler and less featureful)

## Content Guidelines
- Use Thai as much as possible for accessibility
- Introduce English terms that are important for competitive programming career
  - Example: Use "Dynamic programming" instead of "กำหนดการพลวัต"
- Content is based on 3 years of teaching Computer Olympiad camps

## Key Resources to Promote
1. Main grader sites:
   - https://programming.in.th/
   - https://cses.fi/problemset/
2. Useful books:
   - https://cses.fi/book/book.pdf
   - https://www.csc.kth.se/~jsannemo/slask/main.pdf

## Tech Stack
- Keep it minimal and "boring" for low maintenance
- Technologies: HTML, CSS, (htmx optional)
- Deployment: GitHub Pages
- No complex build systems or frameworks needed

## Working with Copilot
- Focus on improving HTML structure and quality
- Ensure consistent navigation between pages
- Apply semantic HTML best practices
- Keep styling minimal and maintainable
- Prioritize content readability and accessibility
