‡∏ñ‡πâ‡∏≤‡πÄ‡∏£‡∏≤ ‡∏à‡∏≥ $\text{Fib}(i)$ ‡∏ó‡∏µ‡πà‡πÄ‡∏Ñ‡∏¢‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡πÑ‡∏ß‡πâ‡πÉ‡∏ô‡∏≠‡∏≤‡∏£‡πå‡πÄ‡∏£‡∏¢‡πå‡∏´‡∏ô‡∏∂‡πà‡∏á ‡πÇ‡∏î‡∏¢‡πÉ‡∏´‡πâ‡πÄ‡∏Å‡πá‡∏ö‡∏Ñ‡πà‡∏≤‡∏Ç‡∏≠‡∏á $\text{Fib}(i)$ ‡πÅ‡∏•‡∏∞‡πÄ‡∏£‡∏µ‡∏¢‡∏Å‡∏Ñ‡πà‡∏≤‡∏à‡∏≤‡∏Å‡∏ó‡∏µ‡πà‡πÄ‡∏Å‡πá‡∏ö‡∏°‡∏≤‡πÑ‡∏î‡πâ ‡πÇ‡∏Ñ‡πâ‡∏î‡∏à‡∏∞‡∏°‡∏µ‡∏•‡∏±‡∏Å‡∏©‡∏ì‡∏∞‡∏î‡∏±‡∏á‡∏ô‡∏µ‡πâ:

vector<int> v(n, -1);
int fib(int i) {
    if(v[i] > -1) return v[i];

Bottom-up

vector<int> v(n, -1);
int fib(int i) {
    for(int j = 0; j < i; j++) {
        if(j == 0 || j == 1) {
            v[j] = j;
        } else v[j] = fib(j-1) + fib(j-2);
    }
    return v[i];
}
